# Ethernet tool

This software was developed to test the Ethernet protocol.
This is a winform application that uses the TCP/UDP/MULTICAST protocol.

The software uses the user's choice and activated the required protocol for sending and receiving data.

## assets
* Displays and records the data received in the socket
* Sends data that the user enters through the GUI using the selected protocol.


## made of
* Net Framework 4.8, C# and WinForms
* Visual Studio 2022
